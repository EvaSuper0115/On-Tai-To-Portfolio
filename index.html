React Weather App

A responsive application built with React to check the weather in 200,000 cities in the world. It automatically navigates users' location to get the weather for them when first loaded into the page*.
*Location needs to be allowed on browser to enable app, while some browsers require adjustment in settings (e.g. safari), the majority of browsers does that by prompting a "allow location access" pop-up, just click "allow".
